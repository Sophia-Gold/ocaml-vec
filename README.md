# ocaml-vec

An example library using [ocaml-rs](https://github.com/zshipko/ocaml-rs) that wraps the Rust `Vec` type for use in OCaml.

## Building

    make

to run the tests:

    make test

or to build manually:

    cargo build
    jbuilder build
    jbuilder runtest


## API

See `src/vec.mli`
